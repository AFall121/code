数据绑定

  1. 文本插值

    数据绑定最基础的形式就是文本插值，使用的是双大括号标签{{}},为Mustache语法（构建HTML的一种语法）,比如下面这种：

    Hello {{name}}

    单次插值 用v-once

    Hello {{name}}

  2. HTML属性

    用v-bind指令

  3. 绑定表达式

    {{ index + 1 }} {{ index == 0 ? 'true' : 'false'}} {{ name.split('').join('-')}}

  4. 过滤器暂时没整懂。。。

    Vue.js允许在表达式后面添加可选的过滤器，以管道符"|"指示。

    {{ name | uppercase }}

  5. 指令

    1.参数

    v-bind指令可以在后面带一个参数，用冒号（:）隔开,src即为参数

    2.修饰符

    修饰符Modifiers是以半角句号.开始的特殊后缀，用于表示指令应该以特殊方式绑定。
